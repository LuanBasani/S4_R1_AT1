# S4_R1_AT1 - PHP Variáveis e Operadores

Desafio Profissional Inicial — Atividade de PHP com variáveis, operadores aritméticos e de comparação.

## O que faz

- Calcula o tempo de existência de uma empresa (ano atual - ano de fundação)
- Calcula o lucro mensal (ganhos - gastos)
- Verifica se o lucro é maior que R$ 20.000

## Tecnologias

- PHP
